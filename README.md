# 🍽️ FoodApp Backend - Node.js API

This is a **Node.js-based RESTful API** for a Food Ordering Application. It includes full backend functionalities such as **User Authentication**, **Restaurant & Food Management**, **Order Placement**, and **Order Tracking**.

---

## 🚀 Features

### 👤 User Authentication
- ✅ Register a new user
- ✅ Login with email & password
- ✅ Secure password hashing
- ✅ JWT-based Authentication
- ✅ Update password
- ✅ Forgot/Reset password
- ✅ Delete account

### 🏢 Restaurants & Categories
- ✅ Get all restaurants
- ✅ Get all food categories

### 🍕 Food Management
- ✅ Create new food items
- ✅ Update food items
- ✅ Delete food items
- ✅ View all food items

### 📦 Orders
- ✅ Place an order
- ✅ Track order status
- ✅ Update order status (admin)

---

## 🛠️ Tech Stack

- **Node.js** (Express.js)
- **MongoDB** + Mongoose
- **JWT** for Authentication
- **bcryptjs** for password hashing
- **dotenv** for environment management
- **Postman** for API testing
